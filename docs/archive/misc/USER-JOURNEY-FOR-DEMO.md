# 商户运营系统 - 演示视频用户旅程脚本

**版本**: v2.4-stable
**日期**: 2026-02-01
**视频时长建议**: 8-12分钟
**目标受众**: 商场运营总监、一线运营经理、商业地产管理者

---

## 📹 视频结构建议

### 开场（30秒）
- 画面：商场实景 + 系统logo
- 旁白：介绍商场运营痛点（商户流失、租金拖欠、运营效率低）
- 切入：展示系统首页

### 主体演示（10分钟）
- **Part 1**: 项目总经理视角（5分钟）
- **Part 2**: 一线运营经理视角（5分钟）

### 结尾（30秒）
- 核心价值总结
- 联系方式/官网

---

## 🎯 Part 1: 项目总经理视角 - "掌控全局，决策有据"

**角色设定**：李总，负责商场B1-B3层18户商户的整体运营

**核心痛点**：
- 每周需要向高层汇报商户经营状况
- 难以快速识别风险商户
- 帮扶效果无法量化
- 缺乏决策数据支持
