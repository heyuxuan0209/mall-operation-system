## 🎬 开始第一段录制！

### 步骤1：启动录制（10秒）

**在QuickTime中操作：**

1. 点击菜单：**文件 → 新建屏幕录制**
   ```
   或使用快捷键: Command + Control + N
   ```

2. 会弹出录制控制面板：
   ```
   ┌──────────────────────────┐
   │  ⚪ [录制] [选项▼]       │
   └──────────────────────────┘
   ```

3. **点击"选项"下拉菜单**，确认设置：
   - ✅ 麦克风：选择"无"（我们用配音）
   - ✅ 显示鼠标点按：建议勾选（方便观众看到点击）
   - ✅ 品质：最大

4. **点击红色录制按钮** ⚪

5. 会提示你选择录制区域：
   - **建议全屏录制**：直接点击屏幕任意位置
   - 或者拖动选择浏览器窗口区域

6. 点击后会有3秒倒计时，然后开始录制
   ```
   3... 2... 1... 开始录制！
   ```

**录制开始后，立即在终端运行：**

## 💡 录制技巧

### 鼠标移动
- ✅ **缓慢清晰**：不要快速移动，让观众看清
- ✅ **有目的性**：指向配音提到的数据
- ✅ **适当停留**：重要数据停留2-3秒

### 滚动页面
- ✅ **平滑滚动**：不要跳跃式滚动
- ✅ **适时停顿**：图表出现后停留展示
- ✅ **配合配音**：配音说到图表时，图表应该在屏幕中央

### 常见问题处理
❓ **配音和操作不同步怎么办？**
→ 没关系！继续录完，后期可以剪辑调整

❓ **操作失误了怎么办？**
→ 继续录制，记录时间点，稍后重录这一段

❓ **配音声音太小？**
→ 停止录制，调大系统音量，重新开始
