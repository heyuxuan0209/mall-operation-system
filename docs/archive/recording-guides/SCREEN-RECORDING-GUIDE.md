### 安装完成后的配置（5分钟）

#### 第1步：首次启动OBS
1. 在"启动台"或"应用程序"中找到"OBS"
2. 首次打开会弹出"自动配置向导"
3. 选择"仅录制，我不打算直播"
4. 点击"下一步" → "应用设置"

#### 第2步：配置录制区域
1. 在OBS主界面，找到"来源"面板（底部）
2. 点击"+"号 → 选择"窗口捕获"或"显示器捕获"
3. 命名为"浏览器窗口"
4. 选择你的Chrome浏览器窗口
5. 点击"确定"

#### 第3步：设置输出格式
1. 点击右下角"设置"
2. 选择"输出" → "录制格式"：MP4
3. "视频比特率"：5000 Kbps（高质量）
4. 点击"应用" → "确定"

## ⭐ 核心功能：放大缩小特定区域

### 方法A：使用OBS的变换功能（推荐）

**实时放大特定区域**：

1. **录制前设置多个场景**
   - 场景1：全屏视图（用于展示整体）
   - 场景2：放大视图（用于展示细节）

2. **创建场景1（全屏）**
   - 来源：窗口捕获
   - 尺寸：填满画布
   - 用于：展示整体页面

3. **创建场景2（放大）**
   - 右键场景1 → "复制"
   - 重命名为"场景2-放大"
   - 右键来源 → "变换" → "编辑变换"
   - **关键设置**：
     - 缩放：200%（放大2倍）
     - 位置：调整到你想展示的区域
   - 点击"完成"

4. **录制时切换场景**
   - 需要全屏时：点击"场景1"
   - 需要放大时：点击"场景2"
   - **快捷键**：可以设置F1=场景1，F2=场景2

**设置快捷键**：
1. OBS菜单 → "设置" → "快捷键"
2. 找到"切换至场景：场景1" → 设置为 `F1`
3. 找到"切换至场景：场景2" → 设置为 `F2`
4. 点击"应用"

**录制流程**：
```
1. 点击"开始录制"
2. 展示全局 → 按F1（全屏场景）
3. 展示细节 → 按F2（放大场景）
4. 来回切换 → F1/F2自由切换
5. 点击"停止录制"
```

### 方法B：使用macOS自带放大功能（最简单）⭐⭐⭐⭐⭐

**无需配置，立即可用！**

#### 启用放大功能
```bash
# 方式1：命令行启用
sudo defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true
sudo defaults write com.apple.universalaccess HIDScrollZoomModifierMask -int 262144

# 方式2：系统设置
# 系统设置 → 辅助功能 → 缩放 → 勾选"使用键盘快捷键缩放"
```

#### 使用快捷键
- **放大**：`Option + Command + =`（等号键）
- **缩小**：`Option + Command + -`（减号键）
- **跟随鼠标放大**：`Option + Command + 8`（开关）

#### 实际操作流程
1. 打开QuickTime Player（macOS自带）
2. 文件 → 新建屏幕录制
3. 开始录制
4. 需要放大时按：`Option + Command + =`
5. 鼠标移到想放大的区域
6. 需要缩小时按：`Option + Command + -`
7. 停止录制

**优点**：
✅ 完全免费，无需安装
✅ 系统级放大，非常流畅
✅ 可以放大任意区域
✅ 学习成本低

**缺点**：
⚠️ 放大效果是全屏的，不是局部
⚠️ 需要手动控制放大倍数

## 📦 方案2：QuickTime（macOS自带）+ 放大功能 ⭐⭐⭐⭐

### 启用放大功能（1分钟）

1. **打开系统设置**
   - 点击左上角苹果图标
   - 选择"系统设置"

2. **启用缩放功能**
   - 点击"辅助功能"
   - 左侧选择"缩放"
   - 勾选"使用键盘快捷键缩放"
   - 勾选"使用滚动手势配合修饰键来缩放"（可选）

3. **设置缩放样式**
   - 缩放样式：选择"画中画"（推荐）或"全屏幕"
   - 画中画优点：放大部分区域，不影响其他内容

### 录制步骤

1. **启动QuickTime Player**
   ```bash
   open -a "QuickTime Player"
   ```

2. **开始屏幕录制**
   - 文件 → 新建屏幕录制
   - 点击录制按钮
   - 选择录制区域（可选全屏或部分区域）

3. **录制时使用放大**
   - 正常讲解时：不放大
   - 需要展示细节时：
     - 按 `Option + Command + 8` 启动放大
     - 鼠标移到想放大的区域
     - 自动放大该区域
     - 按 `Option + Command + =` 增加放大倍数
     - 按 `Option + Command + -` 减少放大倍数
     - 再按 `Option + Command + 8` 关闭放大

4. **停止录制**
   - 点击菜单栏的停止按钮
   - 或按 `Control + Command + Escape`

### 快捷键总结

| 功能 | 快捷键 |
|------|--------|
| 开关放大 | `Option + Command + 8` |
| 放大 | `Option + Command + =` |
| 缩小 | `Option + Command + -` |
| 停止录制 | `Control + Command + Escape` |

## 📦 方案3：Loom（在线录屏）⭐⭐⭐⭐

**特点**：无需安装软件，浏览器即可录制

### 使用方法

1. **访问Loom官网**
   ```bash
   open https://www.loom.com/
   ```

2. **注册/登录**（免费版足够）

3. **安装浏览器扩展**
   - Chrome扩展商店搜索"Loom"
   - 点击"添加到Chrome"

4. **开始录制**
   - 点击浏览器工具栏的Loom图标
   - 选择"Screen + Camera"或"Screen only"
   - 选择录制区域
   - 点击"Start Recording"

5. **放大功能**
   - Loom有内置的"Focus"功能
   - 录制时点击鼠标，会自动高亮和放大点击区域

**优点**：
✅ 无需安装，浏览器即用
✅ 自动保存到云端
✅ 可以分享链接
✅ 内置放大和高亮功能

**缺点**：
⚠️ 免费版有时长限制（5分钟）
⚠️ 需要网络上传
