# 批量巡检模式 - 快速开始

## 🚀 访问地址
```
http://localhost:3000/inspection/batch
```

或从巡检首页点击右上角"批量巡检"按钮

---

## ✨ 核心功能

### 1. 商户快速切换
- **上一家/下一家按钮**: 快速导航
- **侧边栏商户列表**: 点击任意商户跳转
- **自动跳转**: 完成巡检后自动跳到下一家未完成商户

### 2. 草稿自动保存
- **自动保存**: 切换商户时自动保存当前草稿
- **手动保存**: 点击"保存草稿"按钮
- **自动恢复**: 切换回来时自动加载草稿
- **防丢失**: 关闭浏览器后草稿依然保留

### 3. 进度可视化
- **顶部进度条**: 实时显示完成百分比
- **当前位置**: "3/15" 清晰标识
- **状态标识**:
  - ✅ 绿色 = 已完成
  - ⚠️ 橙色 = 有草稿
  - ⭕ 灰色 = 未开始

## 🎯 使用流程

```
1. 打开批量巡检页面
   ↓
2. 第一家商户巡检
   - GPS签到
   - 拍照记录
   - 快速评分
   - 语音/文字备注
   ↓
3. 点击"完成巡检"
   ↓
4. 查看AI反馈
   ↓
5. 自动跳转下一家
   ↓
6. 重复步骤2-5
```

---

## 💡 使用技巧

1. **批量巡检前准备**
   - 按楼层顺序规划路线
   - 确保手机电量充足
   - 开启GPS定位

2. **中途中断**
   - 随时点击"保存草稿"
   - 关闭页面不会丢失数据
   - 下次打开自动恢复

3. **快速操作**
   - 使用预设评分模板
   - 语音笔记比文字更快
   - 拍照时注意人货场分类
