# 商户历史帮扶档案 - 快速开始指南

## 📖 功能简介

商户历史帮扶档案功能为每个商户建立完整的健康度演进追踪系统，帮助运营团队：
- 📊 查看商户健康度的历史变化趋势
- 🔄 追踪风险等级的升降记录
- 📈 评估帮扶措施的实际效果
- 📁 导出完整的档案数据

## 🚀 快速访问

### 方式一：通过健康度监控页面
1. 访问 http://localhost:3000/health
2. 在商户列表中点击任意商户
3. 在右侧详情面板中向下滚动
4. 找到 **"历史帮扶档案"** 部分（紫色图标）

### 方式二：选择高风险商户
推荐查看以下商户的历史档案（历史变化较丰富）：
- **海底捞火锅** (M001) - 高风险商户，有多次改善记录
- **绿茶餐厅** (M005) - 高风险商户，帮扶案例丰富
- **星巴克咖啡** (M002) - 低风险商户，稳定经营

## 📊 核心功能导览

### 1️⃣ 档案摘要 Tab

**包含内容**：
- **当前状态**：风险等级 + 持续天数
- **6项统计指标**：
  - 历史记录数（快照数量）
  - 风险变更次数
  - 帮扶任务数（已完成/总数）
  - 改善次数（绿色，风险降低）
  - 恶化次数（红色，风险升高）
  - 帮扶成功率
- **健康度趋势**：
  - 最高分/最低分 + 日期
  - 平均分
  - 近30天趋势（改善中/下降中/稳定）
- **最长高风险期**：如有则显示橙色提示

**操作**：
- 点击右上角 **"导出"** 按钮可下载完整档案JSON文件

### 2️⃣ 风险时间线 Tab

**展示内容**：
- **垂直时间轴**：按时间倒序排列（最新在上）
- **风险变更事件**：每个卡片显示：
  - 变化类型标签（改善↑/恶化↓/稳定→）
  - 风险等级变化（从XX到XX）
  - 健康度评分变化（分数 + delta）
  - 触发原因（巡检完成/任务完成等）
  - 相对时间（今天、昨天、N天前）

**交互**：
- 点击卡片展开查看详细信息
- 包含关联任务ID、巡检ID（可点击跳转，预留）

**颜色编码**：
- 🟢 绿色 = 风险降低（改善）
- 🔴 红色 = 风险升高（恶化）
- ⚫ 灰色 = 风险稳定

### 3️⃣ 健康度趋势 Tab

**图表功能**：
- **折线图**：展示健康度随时间变化
- **风险等级参考线**：
  - 90分线（无风险 vs 低风险）
  - 80分线（低风险 vs 中风险）
  - 70分线（中风险 vs 高风险）
  - 60分线（高风险 vs 极高风险）
- **事件标记**：在关键时间点显示标记（巡检、任务开始/完成）

**维度切换**：
- 点击上方按钮切换视图：
  - **总分**：健康度总评分
  - **租金缴纳**：蓝色线
  - **经营表现**：绿色线
  - **现场品质**：橙色线
  - **顾客满意度**：紫色线
  - **抗风险能力**：红色线

**时间范围选择**：
- 下拉菜单选择：近30天、近90天、近半年、近一年

**Tooltip详情**：
- 鼠标悬停在数据点上查看：
  - 完整日期
  - 健康度评分
  - 5维度得分
  - 风险等级
  - 关联事件

## 💡 使用场景

### 场景1：评估帮扶效果
**问题**：想知道上个月的帮扶任务是否有效果？

**操作步骤**：
1. 打开商户档案
2. 查看 **档案摘要** → 帮扶成功率指标
3. 切换到 **风险时间线** → 找到任务完成事件
4. 查看前后风险等级变化和评分变化

### 场景2：识别高风险期
**问题**：商户何时陷入过高风险？持续了多久？

**操作步骤**：
1. 打开商户档案
2. 查看 **档案摘要** → 最长高风险期（橙色提示框）
3. 切换到 **健康度趋势** → 选择"近一年"
4. 观察折线图中评分低于60或70的时期

### 场景3：对比维度表现
**问题**：商户哪个维度最薄弱？是否有改善？

**操作步骤**：
1. 打开商户档案
2. 切换到 **健康度趋势** Tab
3. 依次点击5个维度按钮，观察各维度趋势线
4. 找到评分最低或波动最大的维度

### 场景4：导出数据报告
**问题**：需要向上级汇报商户的历史帮扶情况

**操作步骤**：
1. 打开商户档案
2. 点击右上角 **"导出档案"** 按钮
3. 下载JSON文件（包含所有快照、风险变更、统计数据）
4. 可用于进一步分析或存档

## 📋 数据说明

### 当前实施方案
- ✅ **模拟历史数据**：为每个商户生成3-6个月的历史记录
- ✅ **数据点数量**：每个商户20-35个快照
- ✅ **无需后端**：纯前端展示，无需数据库

### 快照触发事件
系统会在以下情况自动创建快照（模拟）：
- 🔍 巡检完成
- 📝 帮扶任务创建
- ✅ 帮扶任务完成
- 📊 手动记录
- ⚠️ 风险等级变化

### 风险等级定义
| 等级 | 分数范围 | 颜色 | 图标 |
|-----|---------|------|------|
| 无风险 | ≥90 | 绿色 | ✓ |
| 低风险 | 80-89 | 黄色 | ⓘ |
| 中风险 | 70-79 | 橙色 | ⚠ |
| 高风险 | 60-69 | 红色 | ⚠ |
| 极高风险 | <60 | 紫色 | ☠ |

## 🎨 界面元素说明

### 档案摘要卡片
- **渐变色头部**（紫色到靛蓝）：突出档案功能
- **网格布局**：清晰展示统计数据
- **颜色编码**：
  - 灰色 = 中性统计
  - 绿色 = 正面指标（改善）
  - 红色 = 负面指标（恶化）
  - 蓝色 = 成功率

### 时间线卡片
- **节点颜色**：对应变化类型（绿/红/灰）
- **相对时间**：更直观（"3天前"而不是"2026-01-29"）
- **可展开设计**：默认简洁，点击查看详情

### 趋势图
- **参考线**：虚线标记风险等级分界
- **双色方案**：
  - 总分 = 深灰色（主要）
  - 选中维度 = 彩色（次要）

## ❓ 常见问题

### Q1: 为什么有些商户快照很少？
A: 这是模拟数据，实际使用时会随着巡检和任务创建逐渐积累更多快照。

### Q2: 数据会保存吗？
A: 当前版本使用模拟数据，刷新页面后数据重新生成。后续版本会实现持久化存储。

### Q3: 如何添加手动快照？
A: 当前版本暂不支持手动添加。后续版本会在档案页面添加"创建快照"按钮。

### Q4: 能否对比两个商户的档案？
A: 当前版本只能单独查看。后续可考虑添加对比功能。
