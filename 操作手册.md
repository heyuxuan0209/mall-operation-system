## 系统简介

### 系统定位

商户智运Agent 是一套专为商场运营人员设计的智能帮扶任务管理系统，帮助您：

- ✅ **实时监控**：掌握所有商户的健康状况
- ✅ **智能预警**：自动识别风险商户
- ✅ **AI推荐**：基于历史成功案例智能推荐帮扶措施
- ✅ **全程追踪**：从措施制定到效果评估的完整闭环
- ✅ **经验沉淀**：成功案例自动归档，形成知识库

### 核心价值

- **提升效率**：从发现风险到派单只需3步
- **提高成功率**：AI推荐基于真实成功案例
- **规范流程**：标准化的帮扶工作流程
- **积累经验**：每次成功都会沉淀为可复用的知识

## 功能模块详解

### 1. 健康度监控

**功能**：实时查看所有商户的健康状况

#### 如何使用

1. 点击左侧菜单 **"健康度监控"**
2. 查看商户列表，每个商户显示：
   - 商户名称和业态
   - 健康度总分（0-100分）
   - 风险等级（高/中/低/无）
   - 五维度雷达图

3. **筛选商户**：
   - 使用顶部搜索框搜索商户名称
   - 使用筛选器按风险等级筛选
   - 使用筛选器按业态筛选

4. **查看详情**：
   - 点击商户卡片查看详细信息
   - 查看五个维度的具体评分
   - 查看改善建议

#### 健康度评分说明

**五大维度**：
- 📊 **收缴健康度**：租金缴纳情况
- 💰 **经营健康度**：营收和客流表现
- 🏪 **现场品质**：店面陈列和环境
- ⭐ **顾客评价**：满意度和口碑
- 🛡️ **抗风险能力**：财务稳定性

**风险等级判定**：
- 🔴 **高风险**：总分 < 60分
- 🟠 **中风险**：总分 60-75分
- 🟡 **低风险**：总分 75-85分
- 🟢 **无风险**：总分 ≥ 85分

---

### 2. 风险与派单

**功能**：自动识别风险商户，快速创建帮扶任务

#### 如何使用

**步骤1：查看风险预警**

1. 点击左侧菜单 **"风险与派单"**
2. 查看顶部统计卡片：
   - 待处理预警数量
   - 高危预警数量
   - 已处理数量
   - 租售比预警数量

3. 查看风险预警列表，每条预警显示：
   - 商户名称
   - 风险类型（租金逾期/营收下滑/租售比过高/顾客投诉）
   - 严重程度（高危/中危/低危）
   - 预警内容

**步骤2：筛选预警**

- 使用搜索框搜索商户名称或预警内容
- 按状态筛选：全部/未处理/已处理
- 按严重程度筛选：全部/高危/中危/低危
- 按风险类型筛选

**步骤3：创建任务**

**方式1：单个派单**
1. 找到需要处理的预警
2. 点击右侧 **"创建任务"** 按钮
3. 填写任务信息：
   - 商户名称（自动填充）
   - 任务标题
   - 任务描述（自动填充预警内容）
   - 优先级（紧急/高/中/低）
   - 负责人
   - 截止日期
4. 点击 **"创建任务"** 完成

**方式2：批量派单**
1. 勾选多个需要处理的预警（左侧复选框）
2. 点击顶部 **"批量派单"** 按钮
3. 查看待派单列表，确认无误
4. 点击 **"确认批量派单"**
5. 系统自动为每个预警创建对应任务

💡 **小贴士**：
- 高危预警会自动分配给运营经理
- 中危预警会自动分配给运营助理
- 批量派单可以一次处理多个商户，提高效率

### 3. 帮扶任务中心

**功能**：管理帮扶任务的完整生命周期

#### 任务生命周期

```
措施制定 → 执行中 → 效果评估 → 结案
```

#### 如何使用

**步骤1：查看任务列表**

1. 点击左侧菜单 **"帮扶任务中心"**
2. 左侧显示任务列表，每个任务显示：
   - 商户名称
   - 风险等级
   - 负责人
   - 当前阶段
   - 创建日期

3. **筛选任务**：
   - 使用搜索框搜索商户名称
   - 按风险等级筛选（全部/高风险/中风险）

4. **选择任务**：
   - 点击任务卡片，右侧显示详情

---

#### 阶段1：措施制定

**目标**：为商户制定帮扶措施

**操作步骤**：

1. **查看商户信息**
   - 商户名称和负责人
   - 当前阶段：措施制定

2. **获取AI推荐**
   - 点击 **"生成推荐"** 按钮
   - 系统分析商户问题，匹配知识库案例
   - 显示3个推荐措施，每个显示：
     - 参考案例名称
     - 匹配度（百分比）
     - 具体措施内容

3. **采纳措施**
   - 点击推荐措施右侧的 **"采纳"** 按钮
   - 措施自动添加到"已采纳措施"列表

4. **手动添加措施**
   - 在底部输入框输入自定义措施
   - 按 **Enter** 键或点击 **"添加"** 按钮
   - 措施添加到列表

5. **管理措施**
   - 点击措施右侧的 **垃圾桶图标** 可删除

6. **开始执行**
   - 确保至少添加了1条措施
   - 点击底部 **"确认方案并开始执行"** 按钮
   - 任务进入执行阶段

💡 **小贴士**：
- AI推荐基于历史成功案例，匹配度越高越值得参考
- 可以混合使用AI推荐和手动输入
- 建议添加2-4条措施，不宜过多

---

#### 阶段2：执行中

**目标**：执行帮扶措施，记录执行过程

**操作步骤**：

1. **查看帮扶措施**
   - 顶部显示所有已制定的措施
   - 可以点击 **"新增措施"** 添加新措施
   - 可以点击 **垃圾桶图标** 删除措施
   - 修改措施会自动记录到执行日志

2. **添加执行记录**
   - 点击 **"添加记录"** 按钮
   - 选择记录方式：
     - **手动填写**：自己输入执行内容
     - **采纳策略**：从知识库选择标准策略
   - 输入执行内容（如："完成服务礼仪培训，全体员工参加"）
   - 点击 **"提交"**

3. **管理执行记录**
   - 每条记录显示：日期、操作人、执行内容
   - 点击 **"编辑"** 可修改记录内容
   - 点击 **"删除"** 可删除记录（需确认）

4. **进入评估**
   - 执行一段时间后（通常2-4周）
   - 点击底部 **"进入效果评估"** 按钮
   - 任务进入评估阶段

💡 **小贴士**：
- 建议每周至少添加1-2条执行记录
- 记录要具体，便于后续复盘
- 可以随时调整措施，系统会自动记录

---

#### 阶段3：效果评估

**目标**：评估帮扶效果，决定下一步行动

**操作步骤**：

1. **查看帮扶前后对比**
   - **左侧雷达图**：显示帮扶前（灰色）和帮扶后（绿色）的健康度对比
   - **右侧核心指标**：显示具体改善数据
     - 客流量变化
     - 月销售额变化
     - 租金缴纳状态
     - 健康度评分

2. **效果判定**
   - **选项1：效果达标**
     - 点击 **"✓ 效果达标"** 按钮
     - 系统显示"帮扶成功"提示
     - 点击 **"结案并沉淀至知识库"**
     - 系统询问是否沉淀案例
     - 确认后，案例自动保存到知识库
     - 任务标记为"已完成"

   - **选项2：效果未达标**
     - 点击 **"✗ 效果未达标"** 按钮
     - 系统显示处置方式选择：
       - **升级帮扶**：提交上级继续帮扶
       - **转招商预警池**：输入原因，转入清退流程
     - 选择后任务进入对应状态

💡 **小贴士**：
- 效果判定要客观，基于数据而非主观感受
- 即使未达标，也要记录经验教训
- 成功案例沉淀后可供其他同事参考

### 4. AI帮扶知识库

**功能**：浏览和应用历史成功案例

#### 如何使用

**步骤1：浏览案例**

1. 点击左侧菜单 **"AI帮扶知识库"**
2. 查看顶部统计：
   - 案例总数
   - 标签数量
   - 行业覆盖
   - 我的收藏

3. 浏览案例网格，每个案例显示：
   - 商户名称/行业
   - 标签（如：营收下滑、服务质量）
   - 症状描述
   - 操作按钮

**步骤2：搜索和筛选**

- 使用搜索框搜索关键词（行业、症状、标签）
- 按行业筛选（如：餐饮-火锅）
- 按标签筛选（如：租金问题）

**步骤3：查看案例详情**

1. 点击案例卡片的 **"查看详情"** 按钮
2. 查看完整案例信息：
   - **标签**：问题分类
   - **症状描述**：商户遇到的问题
   - **问题诊断**：问题根源分析
   - **帮扶策略**：总体策略
   - **具体措施**：详细执行步骤
   - **实施效果**：最终结果

3. 点击右上角 **星标图标** 可收藏案例

**步骤4：应用案例**

1. 在案例详情页点击 **"应用此案例"**
2. 选择目标商户（显示有风险的商户）
3. 查看任务预览：
   - 任务标题
   - 帮扶策略
   - 具体措施
4. 点击 **"创建帮扶任务"**
5. 系统自动创建任务并跳转到任务中心

💡 **小贴士**：
- 优先参考同业态的案例
- 关注匹配度高的案例
- 可以收藏常用案例，方便快速查找

## 典型工作流程

### 场景1：发现风险商户并处理

**背景**：系统自动识别到"海底捞火锅"存在营收下滑风险

**操作流程**：

1. **查看预警**
   - 进入"风险与派单"页面
   - 看到"海底捞火锅"的高危预警
   - 预警内容："经营健康度仅35分，营收持续下滑"

2. **创建任务**
   - 点击"创建任务"
   - 填写信息：
     - 标题："海底捞火锅经营改善帮扶"
     - 负责人："张经理"
     - 截止日期：30天后
   - 提交创建

3. **制定措施**
   - 进入"帮扶任务中心"
   - 选择刚创建的任务
   - 点击"生成推荐"
   - 系统推荐3条措施：
     - "协助优化菜单结构，推出高毛利特色菜品"
     - "开展联合营销活动，提升客流量"
     - "提供运营指导，优化成本结构"
   - 采纳前2条，手动添加1条："协调商场资源支持"
   - 点击"确认方案并开始执行"

4. **执行跟进**
   - 第3天：添加记录"完成菜单优化方案设计"
   - 第7天：添加记录"联合营销活动启动，首日客流增长15%"
   - 第14天：添加记录"营销活动持续进行，营收环比提升8%"
   - 第21天：添加记录"商户反馈良好，客流稳定增长"
   - 第28天：点击"进入效果评估"

5. **效果评估**
   - 查看对比数据：
     - 客流量：+18%
     - 月销售额：+12%
     - 健康度评分：从45分提升到68分
   - 判定：效果达标
   - 点击"结案并沉淀至知识库"
   - 确认沉淀案例
   - 任务完成！

**耗时**：约30天
**结果**：商户经营改善，案例沉淀到知识库

### 场景2：批量处理多个风险商户

**背景**：月度巡检发现5个商户存在不同程度的风险

**操作流程**：

1. **批量派单**
   - 进入"风险与派单"页面
   - 勾选5个风险预警
   - 点击"批量派单"
   - 确认派单
   - 系统自动创建5个任务

2. **分配任务**
   - 高风险商户（2个）→ 运营经理
   - 中风险商户（3个）→ 运营助理

3. **并行处理**
   - 每个负责人进入"帮扶任务中心"
   - 查看自己负责的任务
   - 依次制定措施、执行、评估

**优势**：
- 一次操作处理多个商户
- 自动分配责任人
- 提高工作效率

### 场景3：参考历史案例处理新问题

**背景**："绿茶餐厅"出现顾客投诉问题，需要快速处理

**操作流程**：

2. **查看案例详情**
   - 点击"查看详情"
   - 阅读症状描述、诊断、措施、效果
   - 确认适用于当前情况

3. **应用案例**
   - 点击"应用此案例"
   - 选择"绿茶餐厅"
   - 查看任务预览
   - 点击"创建帮扶任务"

4. **调整和执行**
   - 系统自动创建任务，措施已预填充
   - 根据实际情况微调措施
   - 开始执行

**优势**：
- 站在前人肩膀上，避免重复摸索
- 提高成功率
- 节省时间

## 常见问题

### Q1：如何判断商户是否需要帮扶？

**A**：主要看两个指标：
1. **健康度评分**：低于60分建议介入
2. **风险预警**：系统自动识别的预警都需要关注

### Q2：AI推荐的措施一定要采纳吗？

**A**：不一定。AI推荐仅供参考，您可以：
- 全部采纳
- 部分采纳
- 完全自定义

建议结合实际情况灵活运用。

### Q3：任务可以中途修改措施吗？

**A**：可以！在"执行中"阶段：
- 可以新增措施
- 可以删除措施
- 系统会自动记录所有变更

### Q4：如果帮扶效果不理想怎么办？

**A**：在"效果评估"阶段选择"效果未达标"，然后：
- **升级帮扶**：提交上级继续帮扶
- **转招商**：启动清退流程

### Q5：如何查看我负责的所有任务？

**A**：进入"帮扶任务中心"，任务列表会显示所有任务。
未来版本会增加"我的任务"筛选功能。

### Q6：知识库的案例从哪里来？

**A**：两个来源：
1. **系统预置**：精选的典型成功案例
2. **自动沉淀**：您完成的成功任务会自动归档

### Q7：可以导出数据吗？

**A**：当前版本暂不支持导出。
v2.0 版本会增加Excel导出功能。

### Q8：移动端可以使用吗？

**A**：可以！系统支持响应式设计，在手机和平板上都能正常使用。
建议使用平板或电脑以获得最佳体验。

### Q9：数据会丢失吗？

**A**：当前版本数据存储在浏览器本地（localStorage）。
建议：
- 定期备份重要数据
- 不要清除浏览器缓存
- v2.0 版本会迁移到云端数据库

### Q10：如何提高帮扶成功率？

**A**：建议：
1. **及早介入**：健康度低于70分就开始关注
2. **参考案例**：多看知识库的成功案例
3. **定期跟进**：每周至少添加1次执行记录
4. **数据驱动**：基于数据判断效果，不凭感觉
5. **经验总结**：成功后及时沉淀案例
